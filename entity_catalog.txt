BRONZE AGE TRADERS
ENTITY CATALOG

Purpose:
- Human-editable design catalog for items, NPCs, and related world entities.
- Not loaded automatically by the game yet.
- Intended workflow: edit this file in plain text, then ask Codex to sync the changes into code.

Suggested future workflow:
- Add or modify entries here.
- Keep the entry format and field names readable rather than code-perfect.
- Ask Codex something like: "Load the new entries from entity_catalog.txt into the game."


============================================================
FIELD GUIDE
============================================================

Item fields:
- id: internal item id
- name: display name
- category: gold | usable | trade | equipment | readable | ammo
- slot: none | weapon | ranged | body | feet | head | light
- glyph: map character
- color: rgb floats
- weight_lbs: item weight in pounds
- base_value_g: canonical price in gold
- material: flavor material
- atk_bonus: attack bonus if equipment/ammo
- def_bonus: defense bonus if equipment
- heal: healing dice for usable items
- ammo_type: ammo id consumed by ranged weapon
- range: ranged range in tiles
- stack_size: default ammo stack size
- light_fov: light radius when equipped and lit
- burn_turns: remaining/max fuel lifetime for light sources
- text: readable text body

NPC fields:
- id: internal npc id
- name: display name
- glyph: map character
- color: rgb floats
- str/dex/con/int/wis/cha: ability scores
- base_hp: HP before CON modifier
- defense: base defense
- power: base melee damage bonus
- level: current creature level
- attack_speed: current attack speed scalar
- merchant: yes | no
- wildlife: yes | no
- mountable: yes | no
- on_attacked: flee | retaliate
- buy_mult: merchant buy multiplier
- sell_mult: merchant sell multiplier
- spawn_weight: village spawn weighting
- wander_radius: tile radius around home
- move_chance: wildlife/docile move chance
- dialogue: bump/interaction lines
- trade_stock: merchant inventory
- skin_table: butchering/skinning yields by quality
